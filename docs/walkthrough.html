Walkthrough

Make sure you've installed everything correcly before following this guide, if not, run through the steps in the Installation.

Some knowledge (changing directories, etc.) of using the commandline is assumed.

1. Create a new project

Creating a new project involves downloading the skeleton project into the right directory, and installing the Node.js dependencies.

frntndr project-name

The project name may not contain characters other than a-z0-9, a minus sign: - and a dot: .. The name can not start with a dot however. Read more on naming at npmjs.org.

You could also specify . as the project name, to download the skeleton into the current directory.

2. Start your development server

Make sure you are in your project's directory, then run:

node app

You can now open http://localhost:1337/ in your favourite browser.

If you've got an IDE instead of a text editor, like WebStorm, you could set up a Run Configuration to easily start your development server with a play button from your IDE, instead of using the scary command line.
