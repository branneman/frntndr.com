Installation

1. Install dependencies

1.1. Node.js

First of all, you need Node.js. You'll need at least Node.js version 0.10.x to run frntndr.

Node can be installed in several ways, but to ensure you can swap versions easily, I advise you don't install Node directly, but use a version manager instead. There are several version managers available, such as:

  • nvm (unix) or nvmw (windows)
  • n (unix) or nodist (windows)

It's also great to use a version manager because Node.js releases fast, and frntndr will keep testing on the latest and greatest of the stable branch. Stuff might get added that only works on the newer versions.

If you are not going to use Node.js for anything else right now, directly downloading and installing from the Node.js site will do fine. Updating Node is just more of a drag then.

1.2. Ruby

Ruby is bundled with OSX, so you won't have to install that manually. Windows users have to download Ruby from RubyInstaller.

Once you've got Ruby up and running, install Bundler with this command:

gem install bundler

2. Install frntndr via npm

All you have to do is run this:

npm i -g grunt-cli bower frntndr

From now on, you can create new projects with the frntndr command-line tool. If you are new to frntndr, you should start with the Walkthrough.
