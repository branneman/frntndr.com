frntndr

About

“Skeleton project and workflow for front-end developers.”

frntndr is a skeleton project for modern front-end codebases. It dictates a modular workflow, and is feature-packed with best-practices. It's build on top of Node.js, and has a simple cli tool to help you set up a new front-end codebase from scratch.
